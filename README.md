# README

Jwt Token based rest api implementation.

Things you may want to cover:

* Ruby version 2.5.1

* Rails version 5.2.4

*  install dependeny <br>
   `bundle install` <br>

*  Migrate Table <br>
   `rails db:migrate` <br>

*  Run Rails Server <br>   
   `rails s`
